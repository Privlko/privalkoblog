Transitions in Marriage and Earnings

2020/02/09

I previously write a post about earnings and marriage. There, I briefly talked about the marriage premium and listed some stats from the RLMS dataset. But really, the marriage premium is supposed to work as a dynamic process, where we catch people before and after they get married and then estimate that effect on earnings. I’m still thinking about how best to do this, but there are some tips in Longhi and Nandi’s book on panel analysis. Basically, by using some lags, we can create a measure for changes in marriage status between waves.
